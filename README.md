## О проекте

**LuxTrader** — учебно-портфельный проект: визуальная часть сервиса премиальных аукционов. Одна HTML-страница демонстрирует типичный UX площадки — от hero-блока до каталога лотов, категорий, услуг и информационных блоков.

Контент ориентирован на luxury-сегмент: часы, ювелирные изделия, автомобили, недвижимость, произведения искусства и смежные категории. Все интерактивные сценарии (ставки, личный кабинет, регион) реализованы как **статический макет** — ссылки ведут на `#`, данные лотов и таймеры зафиксированы в разметке.

В репозитории только production-артефакты (`*.min.css`, `*.min.js`); исходники SCSS/JS не включены.

## Особенности реализации

- **Три карусели Swiper** — hero (2 слайда), популярные лоты, блок цитат с кастомными стрелками и декором.
- **Карточки лотов** — таймер аукциона, текущая ставка в ₽, счётчики просмотров и участников, CTA «Поднять ставку».
- **Сетка из 9 категорий** — иконка, фоновое изображение, SVG-декор; ниже — текстовый блок о рынке luxury-аукционов.
- **Блок услуг** — экспертная оценка, заказ комплектующих, ремонт и реставрация.
- **Навигация по секциям** — `data-goto` / `data-goto-header` с учётом высоты шапки; подсветка активного пункта через `data-watch="navigator"`.
- **Адаптив** — бургер-меню, выпадающее меню пользователя (регион, кабинет, ставки, лоты), перенос копирайта в футере через `data-da` (breakpoint 767px).
- **Оптимизация изображений** — `<picture>` (WebP + PNG/JPG), lazy-load с `data-src` / `data-srcset` и placeholder до загрузки.
- **Единая система заголовков** — компоненты `title`, `block-header`, `block-decor` для визуальной связности секций.
- **SEO и превью** — `meta description`, Open Graph, Twitter Card; язык страницы `lang="ru"`.
- **Кэш статики** — query-параметры версии на CSS/JS (`?_v=...`) и favicon (`?v=...`); при обновлении файла увеличьте номер в `index.html`.

## Стек

| Слой | Технологии |
|------|------------|
| Разметка | HTML5, семантика (`header`, `main`, `section`, `article`, `footer`) |
| Стили | CSS3, BEM-подобные классы, mobile-first |

## Быстрый старт

```bash
git clone https://github.com/murodjon-dev-uz/luxtrader.git
cd luxtrader
python3 -m http.server 8000
```

Откройте [http://localhost:8000](http://localhost:8000). Альтернатива: `npx --yes serve .`
